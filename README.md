# 🎬 Movie Library (Личная кинотека)

**Автор:** [Малухов Дамир]
**Язык:** Python 3

## Описание проекта

Простое десктопное приложение для ведения личной коллекции фильмов. Позволяет добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга. Данные сохраняются в файл `movies.json`, чтобы не терялись после закрытия программы.

## Возможности

*   ✅ **Добавление фильмов:** Ввод данных через удобную форму.
*   ✅ **Валидация:** Проверка корректности года (число) и рейтинга (от 0 до 10).
*   ✅ **Фильтрация:** Быстрый поиск по жанру и году выпуска.
*   ✅ **Сохранение/Загрузка:** Все данные хранятся локально в формате JSON.
*   ✅ **История изменений:** Проект готов к работе с системой контроля версий Git.

## Инструкция по запуску

1.  Установите Python 3 (если не установлен).
2.  Скачайте или клонируйте этот репозиторий.
3.  Запустите файл `movie_library.py`:
    ```bash
    python movie_library.py
    ```
4.  Пользуйтесь!
